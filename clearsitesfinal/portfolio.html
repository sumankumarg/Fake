Project Description

We were approached to build a multi-tenant application that would allow a wholesale vendor of jewelery and fashion products to manange inventory, orders and shipping while allowing retail vendors to sell these products through their own e-commerce portals. The retail e-commerce portals needed to behave like completely independent websites with their own customised look and feel.

The solution implemented by us uses REST (or more precisely, POX) to allow any client that has an API key to use the wholesale application's API to list products, create a shopping cart and place orders. Retail e-commerce portals are implemented as web applications that consume the wholesale API, allowing us to spin up new, fully functional retail websites in under thirty minutes.
